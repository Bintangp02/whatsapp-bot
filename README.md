## Clone this project

```bash
> git clone https://github.com/Bintangp02/sticker-bot
```

## Install the dependencies:

```bash
> npm install gify-cli -g
> npm i
```

### Usage
Before running this script, first edit [this section](https://github.com/Bintangp02/sticker-bot/blob/master/msgHndlr.js#L67) with your WhatsApp number, remember your WhatsApp number!  Not a bot number, then
```bash
> npm start
```
